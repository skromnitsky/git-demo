How to transfer all Python Packages from pip to Anaconda (conda) in Windows?

Here's the solution: 1. Enter in cmd: pip freeze

Output: agate==1.6.1 agate-dbf==0.2.1 agate-excel==0.2.3 agate-sql==0.5.4 aiofiles==0.4.0 aiohttp==3.6.2 alabaster==0.7.12 ansi==0.1.3 anvil==0.0.2 appdirs==1.4.3 ...

2. Create requirements.txt

3. If you are using Windows enter in cmd: FOR /F "delims=~" %f in (requirements.txt) DO conda install --yes "%f" || pip install "%f"

That's all. But copying the packages will be very slow...
